# README

## Instructions
In order to run this project, follow the standard rails workflow:
- git clone
- ensure ruby version is set to 2.4.0
- bundle
- rails db:create; db:migrate; db:seed
- rails s
- sign in to localhost:3000
	- demo@demo.com / welcome
